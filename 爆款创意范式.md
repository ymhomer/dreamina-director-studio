# 爆款创意范式  

- **身份穿越 / 平行宇宙**：利用强烈对比将现实与幻想拼接，瞬间抓住眼球。常见做法是让现代角色“穿越”到古代/奇幻场景，或反之，例如将知名IP置入完全不同的文化背景（如瑞克与莫蒂穿越到中国传统场景【20†L15-L18】）。这类创意通常结构化为：**开场钩子**（暗示日常）→**意外转场**（进入新世界）→**情绪高潮**→**记忆点**【35†L158-L164】【20†L15-L18】。例如图中所示，少女从校园一端走入魔法传送门，瞬间从现代校园切换到怪兽城堡，实现了现实与幻想的混搭【20†L15-L18】【35†L158-L164】。  
- **情感反差 / 风格混搭**：通过把截然不同的元素放在一起制造冲突与幽默。例如“古代宫廷人物骑摩托车”“古人蹦迪”“建筑穿毛衣”，都属于古今风格混搭【29†L44-L52】【20†L15-L18】。此类视频开头往往是常见场景（如传统人物、建筑等），随后快速加入违和元素（现代交通、时尚元素等），形成强烈反差，引发惊讶和共鸣【35†L158-L164】【29†L44-L52】。  
- **角色错位 / 超现实效果**：让动物、物品或AI角色扮演人类角色，或反客为主。例如“狗狗当主角驾驶火车”“猫咪拿起话筒唱歌”等。这种错位通过幽默或出人意料的表现激发好奇心，通常以简短剧情或镜头运用来烘托情绪。  
- **满足感/解压题材**：热门“解压”视频通过**物理模拟**带来快感。例如“沉浸式切割”系列，把冰块、金属等物体以极高清画面切碎；“万物皆可捏/毛绒”系列，把日常物品变形示范【28†L40-L48】。这类内容不需复杂剧情，但画面冲击力强，容易产生大量点赞和转发【28†L40-L48】【35†L158-L164】。  
- **时间循环 / 反转剧情**：利用时间回环或反复情节吸引注意。例如让人物不断重复同一天的不同结果，或在结尾给出意料之外的反转结局。此类模式常见于悬疑或搞笑短片，能在短时间内产生强烈的情绪变化。  
- **通用剧情公式**：无论创意类型，成功的AI短视频通常具备几个要素：**丰富的分镜**、**强烈的情绪张力**、**频繁的画面变化**【35†L158-L164】。也就是说，一个开头就抓住观众注意（钩子），中间持续堆叠冲突或转折，引发情感波动，最后留有反差或惊喜的记忆点。

# 专业提示词“魔咒”库（Style Tokens）  

- **影视风格**：可引用具体导演或类型的风格关键词，如“Wes Anderson风格”（对称构图、柔和色调）、“Christopher Nolan风格”（暗黑大气、电影质感）、“Netflix纪实风格”、“IMAX电影级画质”等。这些词能瞬间为画面带来专业感，但过多复杂风格可能导致输出风格分散。Dreamina 官方提示库中也建议使用类似“写实摄影”、“电影风”“受吉卜力启发风格”等关键词【9†L142-L147】来统一画面气质。  
- **画面质感**：关键词如“16mm 胶片颗粒”、“暖钨丝灯光”、“HDR”、“超写实细节”、“柔和模糊”等，可控制画面纹理和清晰度。它们会增强调色板和质感，但某些质感词（如强颗粒或过渡模糊）可能使画面显得噪点或软化，对多帧一致性有挑战。  
- **色彩体系**：使用“多巴胺配色”（高饱和橙紫）、“粉彩色调”、“赛博霓虹色”、“阴郁去饱和”等，可快速设定整体氛围。举例来说，“暖色调/粉彩调色盘”提示会让输出偏暖柔，而“荧光霓虹/冷色调”则渲染冷峻或未来感【9†L154-L157】。统一的色彩提示有助于保持风格一致，否则颜色风格跳变会影响视频连贯性。  
- **运镜语言**：提示词加入“推进镜头（dolly in）”、“手持晃动（handheld shaky cam）”、“无人机视角（FPV drone）”、“长镜头一镜到底（one-shot sequence）”等，可以暗示镜头运动风格。实际效果视模型而定：有时能让动画更具动感，一次稳定拍摄时也可用相机属性词汇（如“过肩视角”“广角”）来指定镜头角度【9†L170-L172】。  
- **艺术流派**：如“水彩画”、“国画/水墨”、“蒸汽波（Vaporwave）”、“赛博朋克风”、“巴洛克风格”、“极简主义”等，都可作为风格锚点【9†L142-L147】。添加这些流派关键词能让模型生成相应的艺术质感画面，但要注意某些风格（如极简主义）可能删减细节，复杂风格（如巴洛克）则输出负荷大。总之，恰当的风格词可提高画面一致性，过度堆砌则会降低稳定性。  

# 提示词结构公式（Prompt Framework）  

- **通用模板**：建议按顺序结构化提示词：**主体 + 动作/姿势 + 场景/环境 + 光影/镜头 + 风格/美感 + 情绪/氛围**。例如：“*年轻女探员在迷雾缭绕的霓虹街头疾奔（主体+动作+场景），城市远处光线昏暗，手持镜头跟拍（光影+镜头），画面采用诺兰式暗色电影风格，整个场景紧张惊险*”。这个结构清晰地将元素拆分，使模型更准确理解需求【9†L116-L125】【9†L127-L130】。  
- **🎬 影视类**：例如剧情片或预告片，可用“**角色+主要行为+电影场景+戏剧性光影+镜头运动+影视风格+紧张/温暖情绪**”的模板。**示例**：“*勇敢的刑警（主体）在废弃工厂追捕匪徒（动作），昏暗车灯光线穿透烟雾（光影），镜头从俯拍推进（镜头），克里斯托弗·诺兰式电影风格，氛围紧张*”。  
- **🎵 音乐类**：可以按“**主唱/乐队+舞台动作+舞台/幻境背景+色彩鲜艳灯光+动感剪辑+音乐流派风格+情绪节奏**”提示。**示例**：“*摇滚歌手（主体）在宇宙星云舞台上狂摇吉他（动作），全景镜头展现绚丽灯光和观众（场景），采用热烈霓虹色调（光影+色彩），快速摇摄镜头，80年代复古风格，激情澎湃*。”  
- **🎨 艺术类**：聚焦画面美感，模板可为“**主体物/角色+静态姿势+背景细节+光影质感+艺术流派+主题色彩+情绪表达**”。**示例**：“*一位古典美女（主体）端坐于花园（场景），金色夕阳柔光照射（光影），画面采用文艺复兴油画风格（风格），淡雅色调下流露庄重宁静*。”  
- **👻 恐怖类**：侧重氛围营造，模板可写“**受害者/主角+危险动作+阴森背景+低调光影+镜头手持+恐怖风格+惊悚情绪**”。**示例**：“*年轻女子（主体）在废弃医院缓步前行（动作），手持晃动镜头跟随其背影（镜头），昏暗走廊墙面斑驳（场景），月光或烛光投射恐怖阴影（光影），哥特式恐怖风格，压抑惊悚*。”  
- **🏙️ 生活类**：真实感日常题材，模板可为“**普通角色+日常行为+生活化场景+自然光/手机镜头+写实风格+轻松情绪**”。**示例**：“*年轻女孩（主体）在阳光咖啡厅摆弄手冲咖啡（动作），镜头近距离记录整个过程（镜头），暖色晨光透过窗户洒入（光影），社交媒体vlog风格，画面真实温馨*。”  
- **🕰️ 复古类**：利用年代感和质感，模板例如“**年代背景+主体动作+旧设备质感+颗粒噪点+陈旧色调+时代风格+怀旧情绪**”。**示例**：“*1960年代汽车工厂（场景），机器人（主体）缓缓组装老式轿车（动作），采用模仿老电视效果的胶片颗粒（光影+风格），画面整体泛黄，怀旧氛围浓厚*。”  

# 质量控制方法（Prompt Engineering 策略）  

- **风格锁定 (Style Lock)**：始终使用一致的参考和关键词锁定主要风格，避免输出偏离。可通过多模态参考（如上传关键参考图片）或在提示词中反复强调风格关键词来实现【12†L55-L59】。例如指定特定角色外观和服装特征，或在多帧提示中重复“文艺复兴油画风格”，能让模型保持统一视觉方向【12†L55-L59】。  
- **负面提示 (Negative Prompt)**：明确告诉模型需要避免的元素。添加否定词可以排除不想要的效果或对象，例如“-模糊 –卡通 –过度饱和”等【13†L278-L283】。正确使用负面提示可去掉干扰项，提高画面纯粹度。例如描述“厨师切肉”的场景时，可加上“无背景人物，无低分辨率噪点”等负面提示，以防出现多余杂物。【13†L278-L283】  
- **角色一致性 (Character Consistency)**：在多帧视频中，保持角色服装、发型和面部特征一致非常关键。提示词中始终使用相同的角色描述或名称，并配合参考图像，有助于模型准确复现同一角色【12†L55-L59】。Dreamina Seedance 2.0支持多参考输入，能够在不同镜头中对同一角色进行精准跟踪和渲染【12†L55-L59】。例如可在每条提示词中加上“同一位女主角，戴着红色连衣裙”，确保连贯性。  
- **物理逻辑约束 (Motion/Physics)**：描述动作时加入物理细节可避免奇怪的运动失真。比如对“高速飞行”“碰撞”等场景，提示中包含“根据现实物理规律”“保持重力/速度一致”等关键词，帮助模型生成更符合逻辑的运动。【12†L61-L64】同时确保音画同步和动作连贯性（Dreamina 已通过对齐视觉和声音防止运动不一致【12†L61-L64】）。例如让人物奔跑，可提示“脚步稳健有力，肌肉运动真实”，避免出现悬浮跳跃或骨骼错位。
